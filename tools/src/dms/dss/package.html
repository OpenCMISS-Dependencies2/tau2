Provides an API for access to TAU performance data.

The DMS package provides access to the TAU data in three forms: PerfDB database, pprof.dat files, and streaming TAU data. The user is expected to create an instance of a DataSession object, and refer to all data through the public methods of the DataSession object.

Currently, only support for PerfDB is available. Support for pprof.dat files and streaming TAU data will be available in a future release.

The DataSession object can return one of several data objects, including Application, Experiment, Trial, Function, FunctionDataObject, UserEvent and UserEventDataObject.
