Introduction

TAU provides a suite of static and dynamic tools that provide graphical user interaction and interoperation to form an integrated analysis environment for parallel C++ applications.

Installation Instructions For POOMA Users

  • Step One: Configure PDT 

    
[FOR SGI]
% cd /usr/local/packages/pdtoolkit-1.1
% ./configure -arch=IRIX64 -KCC
[FOR LINUX PCs]
% ./configure -KCC

  • Step Two: Configure TAU 

    
% cd /usr/local/packages/tau-2.7
[FOR SGI]
% ./configure -arch=sgi64 -c++=KCC -tulipthread=/usr/local/packages/smarts-1.0.1 -smarts -SGITIMERS -pdt=/usr/local/packages/pdtoolkit-1.1 
[FOR LINUX PCs]
% ./configure -arch=linux -c++=KCC -tulipthread=/usr/local/packages/smarts-1.0.1 -smarts -pdt=/usr/local/packages/pdtoolkit-1.1 
% make install

  • Step Three: Configure SMARTS

    
% cd /usr/local/packages/smarts-1.0.1
[FOR SGI]
% configure --with-arch=iris4d --prefix /usr/local/packages/smarts-1.0.1 --with-taudir=/usr/local/packages/tau-2.7 --enable-64bit --enable-profile 
[FOR LINUX PCs]
% configure --with-arch=i386-linux --prefix /usr/local/packages/smarts-1.0.1 --with-taudir=/usr/local/packages/tau-2.7 --enable-profile 
% make
% make install

  • Step Four: Configure Pooma II

    
% setenv TAUDIR     /usr/local/packages/tau-2.7
% setenv PDTDIR     /usr/local/packages/pdtoolkit-1.1
% setenv SMARTSDIR  /usr/local/packages/smarts-1.0.1
[FOR SGI]
% ./configure --arch SGI64KCC --suite PP --parallel --profile --opt -ex
[FOR LINUX PCs]
% ./configure --arch LINUXKCC --suite PP --parallel --profile --opt -ex
% setenv POOMASUITE PP
% make
% cd examples/Solvers/SimpleJacobi
% make
% cd $POOMASUITE
% ./SimpleJacobi
% pprof
% racy
